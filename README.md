# Tidal Commerce Technical Assessment

This project is a modern e-commerce frontend built with Next.js and GraphQL. It includes a product listing page, a product detail page with dynamic routing, and interactive UI components such as modals, image galleries, and variant selection.

## Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Apollo Client (GraphQL)

## Features

* Product list with image, title, and price
* Modal with product details and quantity selector
* Product detail page with image gallery
* Variant selection (size and color)

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open in browser:

```bash
http://localhost:3000
```

## Notes

The project uses the Shopify Storefront GraphQL API for product data. The UI is designed to be modular and easily extensible for additional features such as cart functionality or persistent state.
